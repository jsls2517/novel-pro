# AI痕迹检测指南

## 自动检测维度

### 1. 段落等长检测（dim 20）
**问题**：AI生成的段落长度过于均匀
**检测**：段落长度变异系数 < 0.15
**解决**：
- 短段落（1-3行）用于节奏加速或冲击
- 长段落（5-8行）用于沉浸描写
- 刻意制造段落长度差异

### 2. 套话密度检测（dim 21）
**高频率AI套话**：
- 似乎、可能、或许、大概
- 某种程度上、一定程度上
- 在某种意义上
- 值得注意的是
- 不可否认的是

**阈值**：>3次/千字触发警告
**解决**：用确定性叙述替代模糊表达

### 3. 公式化转折检测（dim 22）
**高频转折词**：
- 然而、不过、与此同时
- 另一方面、尽管如此
- 话虽如此、但值得注意的是

**阈值**：同一段落连续使用>2次
**解决**：
- 删除无意义的转折词
- 用动作暗示转折
- 口语化表达（"哪有那么便宜的事"）

### 4. 列表式结构检测（dim 23）
**问题**：连续使用相同前缀的句子
**示例**：
```
他看了看四周。他拿起了刀。他走向了门口。
```
**解决**：变化句式结构，避免机械重复

### 5. 修辞模式检测（dim 24）
**过度使用的修辞**：
- 比喻：像/如/仿佛（>5次/章）
- 排比：重复结构（>3处/章）
- 反问：难道/怎么可能（>2次/章）
- 夸张：天崩地裂/惊天动地

**解决**：控制修辞密度，多样化表达

## 人工检查清单

- [ ] 段落长度有明显差异
- [ ] 没有连续使用相同句式开头
- [ ] 套话词密度 < 3次/千字
- [ ] 转折自然，不依赖转折词
- [ ] 修辞多样化，不过度堆砌
