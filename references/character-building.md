# 人物塑造方法

## 核心标签 + 反差细节

### 构建方法
1. 动机绑定主线：每个配角的行为动机必须与主线产生关联
2. 核心标签 + 反差细节：让配角活过来
3. 通过事件立人设：禁止通过外貌描写堆砌人设
4. 语言区分度：不同角色的说话方式必须有辨识度
5. 拒绝集体反应：群戏中不写众人齐声惊呼

## 六步走人物心理分析

每个重要角色在关键场景中的行为，必须经过以下六步推导：

1. 当前处境：角色此刻面临什么局面？
2. 核心动机：角色最想要什么？最害怕什么？
3. 信息边界：角色知道什么？不知道什么？
4. 性格过滤：这个角色的性格会怎么反应？
5. 行为选择：基于以上四点，角色会做出什么选择？
6. 情绪外化：这个选择伴随什么情绪？用什么身体语言表达？
