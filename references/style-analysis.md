# 风格指纹分析指南

## 统计维度

### 1. 句长分布
**平均句长**：15-25字为正常范围
**标准差**：
- 标准差大 → 长短句交替，节奏变化丰富
- 标准差小 → 句式单调，AI味重

**优化建议**：
- 短句（5-10字）用于紧张/冲击场景
- 中句（15-25字）用于日常叙述
- 长句（30-50字）用于复杂描写

### 2. 段落长度
**平均段落长度**：80-150字
**范围**：
- 短段落（30-60字）：对话、动作
- 中段落（80-120字）：叙述、过渡
- 长段落（150-250字）：环境、心理

### 3. 词汇多样性（TTR）
**计算公式**：独特词汇数 / 总词汇数
**中文参考值**：
- 0.4-0.5：正常范围
- <0.35：词汇重复，可能AI生成
- >0.6：词汇过于分散，可能不连贯

### 4. 开头模式
**检测**：统计每句前2个字的高频模式
**问题模式**：
- 连续多句以"他"开头
- 连续多句以"然后"开头
- 连续多句以"突然"开头

**优化**：变化主语和开头方式

### 5. 修辞特征
**常见修辞频率**：
- 比喻：2-3处/千字
- 排比：1-2处/章
- 反问：0-1处/章
- 夸张：适度使用

## 风格模仿方法

### 步骤1：分析参考文本
1. 提取统计特征（句长、段落、词汇）
2. 识别高频词汇和句式
3. 记录修辞偏好

### 步骤2：建立风格指南
```
风格指纹示例：
- 平均句长：18字
- 段落长度：100-150字
- 偏好短句开头
- 善用比喻（2处/千字）
- 少用转折词
```

### 步骤3：写作时应用
- 控制句长在目标范围内
- 模仿段落节奏
- 使用相似修辞密度
