# 写后验证清单

## 跨章重复检测

### 1. 开头重复
**问题**：连续多章以相似方式开头
**示例**：
- 第3章：清晨，阳光透过窗帘...
- 第4章：清晨，阳光洒进房间...
- 第5章：清晨，阳光照在桌上...

**解决**：
- 变化时间（清晨/午后/深夜）
- 变化场景（室内/室外/交通工具）
- 变化开头方式（对话/动作/内心）

### 2. 结尾重复
**问题**：连续多章以相似方式结尾
**示例**：
- 连续3章都以"他陷入了沉思"结尾

**解决**：参考hook-techniques.md，多样化结尾钩子

### 3. 场景重复
**问题**：同一类型的场景连续出现
**示例**：
- 连续3章都是"主角在房间里修炼"

**解决**：
- 变化场景容器（房间/街道/野外）
- 变化行动方式（修炼/对话/探索）

## 连续性检查

### 时间线检查
- [ ] 时间流逝合理（没有突然跳跃）
- [ ] 时间标记一致（白天/黑夜匹配）
- [ ] 季节/天气连贯

### 空间连续性
- [ ] 场景转换有过渡
- [ ] 地理位置合理（没有瞬间移动）
- [ ] 环境描写一致

### 人物状态
- [ ] 伤势/状态延续（上章受伤，这章还疼）
- [ ] 情绪状态合理（没有无铺垫的情绪突变）
- [ ] 持有物品一致（没有凭空出现的道具）

## 信息边界检查

### 角色知情检查
- [ ] 角色只知道他在场时发生的事
- [ ] 角色不能预知未来
- [ ] 角色不能读取其他角色的内心

### 读者知情检查
- [ ] 读者比角色多知道（制造紧张）
- [ ] 读者比角色少知道（制造好奇）
- [ ] 信息释放节奏合理

## 节奏检查

### 章节类型分布
- [ ] 不是连续多章都是"过渡章"
- [ ] 不是连续多章都是"冲突章"
- [ ] 高潮/过渡/收束比例合理（约 4:2:2:2）

### 情绪节奏
- [ ] 有张有弛，不是一直高压
- [ ] 情绪变化有触发事件
- [ ] 重大情绪有铺垫
