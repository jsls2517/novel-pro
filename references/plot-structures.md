# 情节结构模板

## 三幕式结构（适合大多数小说）

### 第一幕：设置（25%）
- 第1章：介绍主角日常生活
- 第2章：激励事件（打破现状）
- 第3章：拒绝召唤→决定行动

### 第二幕：对抗（50%）
- 第4-5章：进入新世界，遭遇挑战
- 第6章：中点（重大转折）
- 第7-8章：困难升级，盟友可能背叛

### 第三幕：结局（25%）
- 第9章：最低点，看似失败
- 第10章：高潮（最终对决）
- 终章：结局，展示新常态

## 英雄之旅（适合冒险/奇幻）

1. 平凡世界 → 2. 冒险召唤 → 3. 拒绝召唤
4. 遇见导师 → 5. 跨越门槛 → 6. 考验盟友敌人
7. 接近洞穴 → 8. 苦难煎熬 → 9. 奖赏
10. 返回之路 → 11. 复活 → 12. 满载而归

## 悬疑小说结构

### 第一幕：谜题出现
- 第1章：发现尸体/事件发生（钩子：谁干的？）
- 第2章：侦探接手（钩子：为什么特殊？）
- 第3章：初步调查（钩子：线索指向谁？）

### 第二幕：调查深入
- 第4章：审讯嫌疑人（钩子：谁在撒谎？）
- 第5章：新线索出现（钩子：方向错了？）
- 第6章：第二起事件（钩子：连环案件？）

### 第三幕：真相揭示
- 第8章：重大突破（钩子：信任的人有问题？）
- 第9章：最终对决（钩子：真相是什么？）
- 第10章：案件解决（钩子：真的结束了吗？）

## 言情小说结构

### 第一幕：相遇
- 第1章：介绍主角A
- 第2章：介绍主角B
- 第3章：A和B相遇（第一印象不佳但命中注定）

### 第二幕：发展
- 第4-5章：被迫在一起，发现吸引力
- 第6章：亲密时刻
- 第7-8章：误解/障碍出现
- 第9章：危机，关系破裂

### 第三幕：和解
- 第10章：意识到真爱，克服障碍
- 终章：在一起，展示新生活

## 选择指南

| 故事类型 | 推荐结构 | 章节数 |
|---------|---------|--------|
| 冒险/奇幻 | 英雄之旅 | 8-12章 |
| 侦探/悬疑 | 悬疑结构 | 8-10章 |
| 言情 | 言情结构 | 6-10章 |
| 动作/惊悚 | 快节奏结构 | 5-8章 |
